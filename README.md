# Redes Neuronales

Proyecto educativo para el estudio e implementación de redes neuronales artificiales.

## Descripción

Colección de notebooks Jupyter con análisis y experimentación de diferentes arquitecturas de redes neuronales, incluyendo conceptos fundamentales, algoritmos de entrenamiento y aplicaciones prácticas.

## Requisitos

- Python 3.7+
- Jupyter Notebook
- NumPy
- Pandas
- TensorFlow o PyTorch
- Matplotlib

## Instalación

```bash
git clone https://github.com/A1Daniel1/redesNeuronales.git
cd redesNeuronales
pip install -r requirements.txt
jupyter notebook
```

## Contenido

Los notebooks incluyen temas como:
- Fundamentos de redes neuronales
- Perceptrón y activación
- Propagación hacia adelante y hacia atrás
- Optimización de parámetros
- Aplicaciones y casos de uso
